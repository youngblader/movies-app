## Getting Started

Follow this instruction to run all the system on your machine.

Clone the repo
```
git clone https://github.com/youngblader/movies-app
```

After cloning this repo, open XCode and build project

## Built with
 
* Swift
* RxSwift
* UIKit, SnapKit
* Kingfisher
* MVVM
